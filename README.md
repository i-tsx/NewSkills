# New Skills 🎓

**New Skills** is a Ministry of Education (MOE) test website that predicts the most suitable university fields for 12th-grade students based on their grades. The platform is designed to help students understand which academic paths match their performance and interests.

It features a **React front-end** for a modern, responsive interface and an **Express API back-end** to process student grades and generate field predictions. The project is built to be scalable and maintainable, with the potential to integrate advanced prediction algorithms and databases in the future.

## 🚀 Key Features

- Input 12th-grade student grades
- Predict suitable university fields with confidence scores
- Responsive and user-friendly interface
- Modular and scalable architecture
- Ready for future expansion, including database integration and analytics

Made with ❤️ to help students make informed decisions about their university paths.
